# Icons

Place the following PNG files in this directory:

- `icon16.png` — 16×16
- `icon32.png` — 32×32
- `icon48.png` — 48×48
- `icon128.png` — 128×128

Use the same circuit-leaf design as `web-app/public/favicon.png`, resized.
You can use Figma, Sketch, or any image editor to export at each size.
